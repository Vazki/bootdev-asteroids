# Asteroids

A classic Asteroids game built with Python and Pygame, created as part of the [Boot.dev](https://www.boot.dev) "Build Asteroids using Python and Pygame" course.

## Requirements

- Python 3.13+
- [uv](https://github.com/astral-sh/uv)

## Running the game

```bash
uv run main.py
```

## Controls

| Key | Action |
|-----|--------|
| W | Thrust forward |
| S | Thrust backward |
| A | Rotate left |
| D | Rotate right |
| Space | Shoot |
| Escape | Quit |

## Gameplay

Destroy asteroids by shooting them. Large asteroids split into smaller ones when hit. The game ends when an asteroid collides with the player.
